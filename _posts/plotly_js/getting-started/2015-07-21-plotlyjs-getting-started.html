Getting started: plotly.js

Download

Download the minified plotly.js source code and dependencies.

Include the downloaded scripts before the end of the </head> tag in your HTML document: 

<head>
	<script src="plotly-latest.min.js"></script>
</head>

Download plotly.js

plotly.js CDN

You can also use the ultrafast plotly.js CDN link. This CDN is graciously provided by the incredible team at Fastly. 

<head>
   <script src="https://cdn.plot.ly/plotly-latest.min.js"></script>
</head>

Start plotting!

In your HTML document, create an empty DIV to draw the graph in: 

<div id="tester" style="width:600px;height:250px;"></div>

Now you can make interactive plotly.js charts using Plotly.plot(). 

<script>
	TESTER = document.getElementById('tester');
	Plotly.plot( TESTER, [{
	x: [1, 2, 3, 4, 5],
	y: [1, 2, 4, 8, 16] }], {
	margin: { t: 0 } } );
</script>

Now you can pass Plotly.plot() either the ID of the DIV ("tester") on the DIV DOM element (TESTER).

Webpack user?

If you are a webpack user, you'll notice that you can't immediately require('plotly.js') because of some dependencies that use Browserify transforms (namely, glsify and a few internal parts). Don't worry though! You can get around this by using the distribution version of plotly.js. 


// entry.js

var Plotly = require('plotly.js/dist/plotly.js');

var data = [{
    x: [1, 2, 3],
    y: [4, 5, 2]
}];

var layout = {
    title: 'plotly.js with Webpack'
};

Plotly.plot('graph-div', data, layout);


// webpack.config.js

module.exports = {
    entry: './entry.js',
    output: {
        filename: 'bundle.js'
    }
};


<!-- index.html -->
<html>
    <head>
        <title>
            plotly.js With Webpack
        </title>
    </head>
    <body>
        <script src="bundle.js"></script>
        <script>
            console.log('Using plotly.js version ', Plotly.version);
        </script>
    </body>
</html>

We are aware that this isn't ideal, and will take it into consideration as plotly.js progresses and becomes more modular.
