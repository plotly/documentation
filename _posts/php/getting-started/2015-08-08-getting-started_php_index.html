Getting started: Plotly for PHP

Installation

To install Plotly's PHP simply copy the plotly.php your PATH. 

cp plotly.php <my php directory>

Connecting the API

In your PHP script, include the plotly.php file and initialize your connection! 

require_once(dirname(__FILE__) . '\..\lib\plotly.php');

//  create the plotly obj
$pl_obj = new Plotly();
                        
//  intialize it with your credentials
$pl_obj->initialize( "username", "api_key" );

You'll need to replace "username" and "api key" with your Plotly username and API key.

Find my API key.


Start plotting!

Copy and paste the following to create your first graph using the Plotly PHP library! If all goes well, the url for your graph will be printed , as part of the $res object in your console. 


// create the 'args' param
$a=array( array(0, 1, 2), array(3, 4, 5), array(1, 2, 3), array(6, 6, 5) );

// create the 'kwargs' param
$kw=array( 
    "filename" => "plot from api",
    "fileopt" => "overwrite",
    "style" => array( 
                    "type" => "bar",
                ),
    "traces" => array( 1 ),
    "layout" => array( 
                    "title" => "experimental data"
                ),
    "world_readable" => true
);

// plot a graph
$res = $pl_obj->post( "plot", $a, $kw );

//
var_dump( $res );
