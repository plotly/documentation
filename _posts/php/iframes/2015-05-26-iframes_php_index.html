Plotly graphs can be embedded in any HTML page. This includes IRuby notebooks, Wordpress sites, dashboards, blogs, and more.


For more on embedding Plotly graphs in HTML documents, see our tutorial.


From Ruby, you can generate the HTML code to embed Plotly graphs with the plotly.tools.get_embed function.



import plotly.tools as tls

tls.get_embed('https://plot.ly/~chris/1638')
