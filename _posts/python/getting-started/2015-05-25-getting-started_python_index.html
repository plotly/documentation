Getting Started: Plotly for Python

{% include python-getting-started.html %}

Start plotting!

Fire up Python! 

$ python

Copy and paste the following to create your first Plotly graph using the Plotly Python library: 

import plotly.plotly as py
from plotly.graph_objs import Scatter

trace0 = Scatter(x=[1, 2, 3, 4],
    y=[10, 15, 13, 17]
)
trace1 = Scatter(
    x=[1, 2, 3, 4],
    y=[16, 5, 11, 9]
)
data = [trace0, trace1]

unique_url = py.plot(data, filename = 'basic-line')

You can also create plotly graphs with matplotlib syntax. Learn more in our matplotlib documentation.

You can also create plotly graphs within an IPython notebook. Learn more in our IPython notebook documentation.

By default, plotly graphs are public and saved to your online plotly account. To create graphs offline and locally, license plotly offline. To create private graphs, set sharing to 'private': 

py.plot(data, share='private', filename='private-plot') # you'll need to be logged in to plotly to view this graph
or, create a randomized "secret" URL:
py.plot(data, share='secret', filename='secret-plot') # anybody with the URL can view the plot
