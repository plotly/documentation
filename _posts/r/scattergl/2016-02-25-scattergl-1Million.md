# WebGL with 1 Million points 


```r
library(plotly)

n <- 1e6
x <- rnorm(n)
y <- 2*x + rnorm(n, sd = 5)

p <- plot_ly(x = x, y = y, alpha  = 0.01) %>% toWebGL()

# Create a shareable link to your chart
# Set up API credentials: https://plot.ly/r/getting-started
chart_link = plotly_POST(p, filename="webgl/1e6")
chart_link
```
