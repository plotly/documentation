### Polar Scatter Chart


```r
library(plotly)
p <- plot_ly(
  plotly::mic, r = ~r, t = ~t, color = ~nms, alpha = 0.5, type = "scatter"
)
layout(p, title = "Mic Patterns", orientation = -90)
```

### Polar Area Chart


```r
p <- plot_ly(plotly::wind, r = ~r, t = ~t) %>% add_area(color = ~nms)
layout(p, radialaxis = list(ticksuffix = "%"), orientation = 270)
```
