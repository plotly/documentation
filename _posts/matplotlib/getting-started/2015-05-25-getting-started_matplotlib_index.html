Getting started: Plotly for Matplotlib

{% include python-getting-started.html %}

Start plotting!

Make a graph using Matplotlib syntax: 

import matplotlib.pyplot as plt
import plotly.plotly as py

mpl_fig = plt.figure()
# --> your matplotlib methods <--

unique_url = py.plot_mpl(mpl_fig, filename="plotly version of an mpl figure")

Alternatively, convert matplotlib figures into Plotly figure objects with mpl_to_plotly: 

import matplotlib.pyplot as plt
import plotly.tools as tls

mpl_fig = plt.figure()
# --> your matplotlib methods <--

plotly_fig = tls.mpl_to_plotly(mpl_fig)

unique_url = py.plot(plotly_fig)
