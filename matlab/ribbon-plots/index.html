    Ribbon Plots in MATLAB®

    How to make Ribbon Plots in MATLAB® with Plotly.


    MATLAB®

    Ribbon Plot

    Create a ribbon plot of the peaks function.

    [x,y] = meshgrid(-3:.5:3,-3:.1:3);
z = peaks(x,y);

figure
ribbon(y,z)

fig2plotly()

    plot00ribbonplot

    [x,y] = meshgrid(-3:.5:3,-3:.1:3);
z = peaks(x,y);
figure
ribbon(y,z)
fig2plotly()
