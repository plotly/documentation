    Network Graphs in MATLAB®

    How to make Network Graphs plots in MATLAB® with Plotly.


    Graphing Network Graphs with Adjacency Matrices

        [B, V] = bucky;
    H = sparse(60, 60);
    k = 31:60;
    H(k, k) = B(k, k);

    fig = figure;
    gplot(B - H, V, 'b-');
    hold on;

    gplot(H, V, 'r-');
    axis off equal;

    fig2plotly(fig);
