    Histograms in R

    How to make a histogram in R.

    • Python
    • R
    • JavaScript
    New to Plotly?

    Plotly is a free and open-source graphing library for R. We recommend you read our Getting Started guide for the latest installation or upgrade instructions, then move on to our Plotly Fundamentals tutorials or dive straight in to some Basic Charts tutorials.

    Basic Histogram

    library(plotly)
p <- plot_ly(x = ~rnorm(50), type = "histogram")

p

    Normalized Histogram

    library(plotly)
p <- plot_ly(x = ~rnorm(50),
             type = "histogram",
             histnorm = "probability")

p

    Specify Binning Function

    library(plotly)

x = c("Apples","Apples","Apples","Organges", "Bananas")
y = c("5","10","3","10","5")

p <- plot_ly(y=y, x=x, histfunc='sum', type = "histogram") %>%
  layout(yaxis=list(type='linear'))

p

    Horizontal Histogram

    library(plotly)
p <- plot_ly(y = ~rnorm(50), type = "histogram")

p

    Overlaid Histograms

    p <- plot_ly(alpha = 0.6) %>%
  add_histogram(x = ~rnorm(500)) %>%
  add_histogram(x = ~rnorm(500) + 1) %>%
  layout(barmode = "overlay")

p

    Stacked Histograms

    p <- plot_ly(alpha = 0.6) %>%
  add_histogram(x = ~rnorm(500)) %>%
  add_histogram(x = ~rnorm(500) + 1) %>%
  layout(barmode = "overlay")

p

    Cumulative Histogram

    library(plotly)
p <- plot_ly(x = ~rnorm(50),
             type = "histogram",
             cumulative = list(enabled=TRUE))

p
