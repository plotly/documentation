    Bar Charts in Julia

    How to make a bar chart in julia. Seven examples of grouped, stacked, overlaid, and colored bar charts.

    • Python
    • R
    • JavaScript

    Basic Bar Chart

    # Learn about API authentication here: https://plot.ly/julia/getting-started
# Find your api_key here: https://plot.ly/settings/api

using Plotly


data = [
  [
    "x" =&gt; ["giraffes", "orangutans", "monkeys"],
    "y" =&gt; [20, 14, 23],
    "type" =&gt; "bar"
  ]
]
response = Plotly.plot(data, ["filename" =&gt; "basic-bar", "fileopt" =&gt; "overwrite"])
plot_url = response["url"]

    Grouped Bar Chart

    # Learn about API authentication here: https://plot.ly/julia/getting-started
# Find your api_key here: https://plot.ly/settings/api

using Plotly


trace1 = [
  "x" =&gt; ["giraffes", "orangutans", "monkeys"],
  "y" =&gt; [20, 14, 23],
  "name" =&gt; "SF Zoo",
  "type" =&gt; "bar"
]
trace2 = [
  "x" =&gt; ["giraffes", "orangutans", "monkeys"],
  "y" =&gt; [12, 18, 29],
  "name" =&gt; "LA Zoo",
  "type" =&gt; "bar"
]
data = [trace1, trace2]
layout = ["barmode" =&gt; "group"]
response = Plotly.plot(data, ["layout" =&gt; layout, "filename" =&gt; "grouped-bar", "fileopt" =&gt; "overwrite"])
plot_url = response["url"]

    Colored and Styled Bar Chart

    # Learn about API authentication here: https://plot.ly/julia/getting-started
# Find your api_key here: https://plot.ly/settings/api

using Plotly


trace1 = [
  "x" =&gt; [1995, 1996, 1997, 1998, 1999, 2000, 2001, 2002, 2003, 2004, 2005, 2006, 2007, 2008, 2009, 2010, 2011, 2012],
  "y" =&gt; [219, 146, 112, 127, 124, 180, 236, 207, 236, 263, 350, 430, 474, 526, 488, 537, 500, 439],
  "name" =&gt; "Rest of world",
  "marker" =&gt; ["color" =&gt; "rgb(55, 83, 109)"],
  "type" =&gt; "bar"
]
trace2 = [
  "x" =&gt; [1995, 1996, 1997, 1998, 1999, 2000, 2001, 2002, 2003, 2004, 2005, 2006, 2007, 2008, 2009, 2010, 2011, 2012],
  "y" =&gt; [16, 13, 10, 11, 28, 37, 43, 55, 56, 88, 105, 156, 270, 299, 340, 403, 549, 499],
  "name" =&gt; "China",
  "marker" =&gt; ["color" =&gt; "rgb(26, 118, 255)"],
  "type" =&gt; "bar"
]
data = [trace1, trace2]
layout = [
  "title" =&gt; "US Export of Plastic Scrap",
  "xaxis" =&gt; ["tickfont" =&gt; [
      "size" =&gt; 14,
      "color" =&gt; "rgb(107, 107, 107)"
    ]],
  "yaxis" =&gt; [
    "title" =&gt; "USD (millions)",
    "titlefont" =&gt; [
      "size" =&gt; 16,
      "color" =&gt; "rgb(107, 107, 107)"
    ],
    "tickfont" =&gt; [
      "size" =&gt; 14,
      "color" =&gt; "rgb(107, 107, 107)"
    ]
  ],
  "legend" =&gt; [
    "x" =&gt; 0,
    "y" =&gt; 1.0,
    "bgcolor" =&gt; "rgba(255, 255, 255, 0)",
    "bordercolor" =&gt; "rgba(255, 255, 255, 0)"
  ],
  "barmode" =&gt; "group",
  "bargap" =&gt; 0.15,
  "bargroupgap" =&gt; 0.1
]
response = Plotly.plot(data, ["layout" =&gt; layout, "filename" =&gt; "style-bar", "fileopt" =&gt; "overwrite"])
plot_url = response["url"]

    Bar Chart with Hover Text

    # Learn about API authentication here: https://plot.ly/julia/getting-started
# Find your api_key here: https://plot.ly/settings/api

using Plotly


data = [
  [
    "x" =&gt; ["Liam", "Sophie", "Jacob", "Mia", "William", "Olivia"],
    "y" =&gt; [8.0, 8.0, 12.0, 12.0, 13.0, 20.0],
    "text" =&gt; ["4.17 below the mean", "4.17 below the mean", "0.17 below the mean", "0.17 below the mean", "0.83 above the mean", "7.83 above the mean"],
    "marker" =&gt; ["color" =&gt; "rgb(142, 124, 195)"],
    "type" =&gt; "bar"
  ]
]
layout = [
  "title" =&gt; "Number of graphs made this week",
  "font" =&gt; ["family" =&gt; "Raleway, sans-serif"],
  "showlegend" =&gt; false,
  "xaxis" =&gt; ["tickangle" =&gt; -45],
  "yaxis" =&gt; [
    "zeroline" =&gt; false,
    "gridwidth" =&gt; 2
  ],
  "bargap" =&gt; 0.05
]
response = Plotly.plot(data, ["layout" =&gt; layout, "filename" =&gt; "bar-with-hover-text", "fileopt" =&gt; "overwrite"])
plot_url = response["url"]
